The Form component is a wrapper for form submissions.

```jsx
<React.Fragment>
  <Form
      onSubmit={()=> console.log('onSubmit')}
      spacing="default"
    >
      <TextField
        required
        name="nameInput"
        id="nameInput"
        labelText="Name"
        value="Johannes Bugiel"
        helpText="Enter name"
      />
      <TextField
        required
        name="emailInput"
        id="emailInput"
        labelText="E-Mail"
        value="johannes.bugiel@contentful.com"
        helpText="Enter email address"
      />
      <Button>Submit</Button>
    </Form>
</React.Fragment>
```

## Best practices <a name="best-practices" />

- The `Form` component wraps content with the HTML <form> component
- To maintain consistency, use the `Form` component to wrap all form input components
- To imitate native HTML behavior, add handler for `onSubmit` action

## Writing guidelines <a name="copy-considerations" />

- Do not use punctuation following a label
- Use sentence style caps (in which only the first word of a sentence or term is capitalized) on labels
