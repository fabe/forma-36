## Best practices <a name="best-practices" />

- Position buttons consistently in the interface
- Use right Icons to represent right actions
- Use extablished Icons
- Pair Icons with text

## Variations <a name="variations" />

Icon component can be configurated in different ways. Here are some examples.

### Colors (`color`) <a name="colors" />

```jsx
<React.Fragment>
  <Icon color="primary" icon="Edit"/>
  <Icon color="positive" icon="Edit"/>
  <Icon color="negative" icon="Edit"/>
  <Icon color="warning" icon="Edit"/>
  <Icon color="secondary" icon="Edit"/>
  <Icon color="muted" icon="Edit"/>
  <Icon color="white" icon="Edit"/>
</React.Fragment>
```
